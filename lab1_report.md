# Отчёт по лабораторной работе №1

## «Создание первого Telegram-бота без программирования»

---

## 1. Описание задачи

### Какую проблему решает бот

Многие люди постоянно ищут, что посмотреть: спрашивают друзей, листают подборки, забывают про фильмы, которые хотели посмотреть. MovieTracker решает эту проблему — позволяет искать фильмы и сериалы, сохранять их в личный список и оценивать просмотренное прямо в Telegram, который и так всегда под рукой.

### Почему выбрали именно эту задачу

Из предложенных вариантов (помощник для команды, сбор фидбека, ассистент задач, агрегатор) решили сделать **свой вариант** — трекер фильмов. Причины:
- Это реально полезный бот, которым хочется пользоваться самому
- Есть бесплатный API (TMDB) с огромной базой фильмов
- Удобно расширять: в Lab2 — подключение API и БД, в Lab3 — деплой и фидбек

## 2. Промпт для LLM

### Исходный промпт

```
Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot (версия 20+, async).

Бот — трекер фильмов и сериалов «MovieTracker»:
- /start — приветствие с описанием бота
- /help — список всех команд
- /search <название> — поиск фильмов (пока заглушка, API подключим позже)
- /popular — список популярных фильмов (пока захардкоженный)
- /trending — тренды дня (пока захардкоженный)
- /top — лучшие по рейтингу (пока захардкоженный)

Требования:
- Использовать async (python-telegram-bot v20)
- Токен хранить в .env через python-dotenv
- Добавить логирование
- Обработка ошибок

Создай:
1. bot.py
2. requirements.txt
3. .env.example
4. README.md
```

### Итерации и улучшения

**Итерация 1:** LLM сгенерировал рабочего бота, но все команды возвращали простой текст. Меню /start — просто список команд текстом.

**Итерация 2:** Попросил добавить inline-клавиатуру в /start с кнопками: Поиск, Популярное, Тренды, Лучшее, Помощь. Бот стал гораздо удобнее — не нужно вводить команды руками.

**Итерация 3:** Попросил обрабатывать нажатия кнопок через `CallbackQueryHandler` — при нажатии на кнопку меню бот показывает соответствующий контент прямо в том же сообщении через `edit_message_text`.

### Финальный промпт

```
Добавь в /start inline-клавиатуру (InlineKeyboardMarkup) с кнопками:
«🔍 Поиск», «🔥 Популярное», «📈 Тренды», «🏆 Лучшее», «📋 Буду смотреть»,
«⭐ Просмотрено», «📖 Помощь».
При нажатии — показывать контент через CallbackQueryHandler + edit_message_text.
Для /help — использовать parse_mode="Markdown" с жирным и курсивом.
```

## 3. Стек технологий

| Технология | Назначение |
|---|---|
| Python 3.11 | Язык программирования |
| python-telegram-bot 20.7 | Библиотека для Telegram Bot API (async) |
| python-dotenv 1.0.1 | Загрузка токена из .env |

### Почему выбрали именно их

- **python-telegram-bot** — самая популярная Python-библиотека для Telegram, хорошая документация, поддержка inline-клавиатур «из коробки».
- **python-dotenv** — безопасное хранение токена, чтобы он не попал в Git.

## 5. Трудности и решения

| Проблема | Решение |
|---|---|
| `RuntimeError: This event loop is already running` | Добавил `close_loop=False` в `app.run_polling()` |
| Кнопки inline-клавиатуры не реагировали | Забыл зарегистрировать `CallbackQueryHandler` в `app.add_handler()` |
| При нажатии кнопки Telegram показывал «часики» | Нужно обязательно вызывать `await query.answer()` в обработчике |
