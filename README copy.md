# MovieTracker Bot 🎬

Telegram-бот для отслеживания фильмов и сериалов. Ищи, сохраняй, оценивай!

Создан в рамках курса «Vibe Coding: AI-боты для бизнеса» (ИТМО).

## Возможности

- Поиск фильмов и сериалов через TMDB API
- Каталог: популярное, тренды дня, лучшее по рейтингу
- Личный список «Буду смотреть» с inline-кнопками ➕
- Оценка фильмов (1-10) с переносом в «Просмотрено»
- Сбор обратной связи и статистика использования
- Inline-клавиатура для быстрой навигации

## Быстрый старт

1. Получите API-ключ TMDB: https://www.themoviedb.org/settings/api
2. Создайте бота через [@BotFather](https://t.me/BotFather)
3. Скопируйте шаблон переменных:
   ```bash
   cp .env.example .env
   ```
4. Впишите `BOT_TOKEN` и `TMDB_API_KEY` в `.env`
5. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
6. Запустите:
   ```bash
   python bot.py
   ```

## Запуск через Docker

```bash
docker-compose up -d
```

## Деплой на Railway

1. Залейте проект на GitHub
2. Подключите репозиторий в [Railway](https://railway.app)
3. Добавьте переменные `BOT_TOKEN` и `TMDB_API_KEY`
4. Railway автоматически определит `Procfile` и запустит бота

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Главное меню с кнопками |
| `/search <название>` | Поиск фильма или сериала |
| `/popular` | Популярные фильмы |
| `/trending` | Тренды дня |
| `/top` | Лучшие по рейтингу |
| `/watchlist` | Список «Буду смотреть» |
| `/watched` | Просмотренное с оценками |
| `/rate <#id> <1-10>` | Оценить фильм |
| `/remove <#id>` | Убрать из списка |
| `/feedback <текст>` | Обратная связь |
| `/stats` | Статистика бота |
